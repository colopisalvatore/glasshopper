# Security Policy

## Supported versions

Only the latest minor release receives security fixes during the pre-1.0 phase.

## Reporting a vulnerability

Please do **not** open a public issue. Email security reports to: `security@TODO.dev` (replace with real address before going public).

Include:
- A description of the issue and the affected component.
- Steps to reproduce.
- Impact assessment if known.

You should receive an acknowledgement within 7 days. We aim to release a fix within 30 days of confirmation, coordinated with you on disclosure timing.
